O **Ybyrá Casting** é uma plataforma institucional focada na curadoria e desenvolvimento de talentos para o setor cultural e audiovisual. O projeto conecta artistas e produtores através de uma gestão eficiente de perfis, agendas e suporte à produção.

[🌐 Acesse o Site Oficial](https://wi3li.site/)

---

## 🚀 Tecnologias e Ferramentas

O projeto utiliza uma stack focada em performance, responsividade e containerização:

* **Frontend:** HTML5, CSS3, JavaScript (ES6+), [Bootstrap 5](https://getbootstrap.com/).
* **Iconografia:** Font Awesome 6.
* **Infraestrutura:** Docker & Docker Compose.
* **Servidor Web:** Nginx (configurado para produção).
* **Integração:** API de métricas personalizada para rastreamento de conversão.
